This utility merges multiple origin annotations into one file. It integrates GNPS annotations (FBMN), SIRIUS, Moldiscovery, and Dereplicator+. Annotations priority is set in this order: First, Experimental match "GNPS/Spectral Match" (Metabolomics Standard Initiative (MSI) level 2), second, High-confidence consensus (MSI level. 3) agreement between Moldiscovery and SIRIUS "Consensus (molD + Sirius)", third, "Moldiscovery" or "Sirius", depending on precursor mass: 1.
